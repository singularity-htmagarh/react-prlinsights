API Integration Examples

Simple, powerful APIs that integrate seamlessly with your existing technology stack

Authentication

POST 
curl -X POST https://api.datainsights.pro/v1/auth \
  -H "Content-Type: application/json" \
  -d '{
    "client_id": "your_client_id",
    "client_secret": "your_client_secret"
  }'

Response:

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600
}

Data Upload

POST 
curl -X POST https://api.datainsights.pro/v1/data \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "dataset": "marketing_campaigns",
    "data": [
      {
        "campaign_id": "12345",
        "spend": 10000,
        "impressions": 500000,
        "clicks": 2500,
        "conversions": 125,
        "date": "2024-01-01"
      }
    ]
  }'

Get Analytics

GET 
curl -X GET https://api.datainsights.pro/v1/analytics/attribution \
  -H "Authorization: Bearer {token}" \
  -G -d "campaign_id=12345" \
  -d "date_range=30d"

Response:

{
  "attribution_model": "data_driven",
  "touchpoints": [
    {
      "channel": "google_ads",
      "attribution_credit": 0.35,
      "conversions": 43.75
    },
    {
      "channel": "facebook_ads",
      "attribution_credit": 0.28,
      "conversions": 35.00
    }
  ],
  "total_conversions": 125
}
